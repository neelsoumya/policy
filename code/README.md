# Code for blogs

## Installation

```bash
python -m venv venv_viz
source venv_viz/bin/activate
pip install -r requirements.txt
```

## Files

* `agi_harms_benefits.py`

Produces plot `ai_xkcd_plot.png`

* `interactive_agi_simulator.py`

Interactive plot with sliders using `ipywidgets`

* `r_code_shiny_rmarkdown.R` and `r_code_shiny_rmarkdown.Rmd`

R markdown and shiny code for AGI policy simulation
